## 🛠️ 快速开始
### ⛳运行环境
- Python 3.7+
- Node.js 18+

### 🎯安装依赖
```
pip install -r requirements.txt
npm install
```

### 🎨配置文件
- 配置文件在.env文件中，将自己的登录cookie放入其中。

### 🚀运行项目
```
python main.py
```
